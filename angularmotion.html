Angular Displacement
Motion in circles: radians

A radian is the measure of an angle created by a radius and an arc length of the same length.



1 full turn around a circle:

- Total displacement = 0
- Δd = 2πr
- Δθ = 360

From the statements above, this proportion can be used:

Arc length/2πr = θ/360

Which leads to these formulas:

Arc length = (θ/360) * 2πr [m]

θ = arc length / radius


Angular Displacement:

Δθ > 0 ..... Motion is counter clockwise
Δθ < 0 ..... Motion is clockwise
